# Tower of Hanoi - JavaFX Implementation

A complete, production-ready implementation of the classic Tower of Hanoi puzzle game using Java and JavaFX.

## Features

### Core Requirements âœ“
- **Stack Implementation**: Each tower uses `Stack<Integer>` for disk management
- **Queue Implementation**: Move logging system using `Queue<String>` 
- **Game Logic**: Complete rule enforcement and win condition detection
- **JavaFX GUI**: Responsive interface with smooth animations

### Advanced Features
- **Hint System**: Provides next optimal moves and strategy guidance
- **Move History**: Complete logging of all game moves
- **Responsive Design**: Scales appropriately for different disk counts

## Technology Stack
- **Java**: JDK 25 (with preview features enabled)
- **JavaFX**: Version 25 for GUI
- **Maven**: Build automation and dependency management

## Building and Running

### Prerequisites
- Java JDK 25
- Maven 3.9.5+

# Package the application
.\mvnw.cmd clean compile

# Run the application
.\mvnw.cmd javafx:run
